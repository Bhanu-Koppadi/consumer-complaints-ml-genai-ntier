### ðŸ“ Description
The **Consumer Complaints ML + GenAI System** is an enterprise-grade solution designed to automate the ingestion, classification, and explanation of textual consumer complaints. By leveraging a hybrid AI architecture, the system combines the deterministic accuracy of **Machine Learning (ML)** with the interpretative power of **Generative AI (GenAI)**.

The system is built on a robust **five-layer N-Tier architecture**, ensuring a clean separation between the presentation, business logic, AI/ML processing, data access, and persistent storage layers. This modular design facilitates easy maintenance, scalability, and adherence to professional software engineering standards.

### ðŸš€ Key Features

- **Persistent Audit Trail:** Stores every complaint, classification, and explanation in a PostgreSQL database.
